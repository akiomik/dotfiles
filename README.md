dotfiles
========

## Abstract
my dotfiles for develop environment.

## How to use

### dotfiles

1. clone this repository.

  ```sh
git clone git@github.com:akiomik/dotfiles.git
  ```

2. create symlinks.

  ```sh
cd dotfiles
./configure
  ```

### Brewfile

1. install brewdler

  ```sh
gem install brewdler
  ```

2. install brew packages

  ```sh
brewdle install
  ```
